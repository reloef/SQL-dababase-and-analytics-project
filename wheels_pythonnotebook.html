plt.bar(x,ybad,0.4, bottom = yvbad,label = 'bad')
plt.bar(x,yokay,0.4, bottom = ybad+yvbad, label = 'okay')
plt.bar(x,ygood,0.4, bottom = yokay+ybad+yvbad, label = 'good')
plt.bar(x,yvgood,0.4, bottom = ygood+yokay+ybad+yvbad, label = 'very good')

plt.ylabel('Customer Feedbaack')
plt.legend()
plt.show()

OBSERVATIONS

  1. Very good and good sentiments is decreasing as quarter number increase
  2. Okay sentiment is not showing much change as quarter increase
  3. bad and very bad sentiment increasing as quarter number increase
  4. This shows an overall decline in custmore satisficaton.
